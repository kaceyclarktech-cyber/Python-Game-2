# Python-Game-2
Number Guessing Game
A simple, fun, interactive Python game where the player tries to guess a randomly generated number.
The game provides hints (too high/ too low) until the correct number is found. 
# Features
Random number generation, unlimited guesses, Too high / too low hints, input validation (numbers only), Beginner-friendly Python logic, clean, readable code.
# How to Play
Run the game in your terminal, the computer chooses a number (default: 1-10), enter your guess, The game tells you: too low , too high, Correct!, Keep guessing until you win.
# Code Overview
The game uses: random.randint() to generate the secret number, a while loop keeps the game running, try/except for safe input handling, simple comparison logic for hints.
